## Key Context

- **V1 positioning:** "Stop Clicking. Start Connecting." — framed Cura as an AI-first ATS
- **V2 positioning:** "Your AI Recruiter. Ready in 5 Minutes." — frames Cura as an AI team member (inspired by nume.ai's "AI CFO" framing)
- **Primary selling point:** Set up an AI recruiter in 5 minutes that does outreach for you
- **Secondary selling point:** Auto CRM update so recruiters focus on relationships
- **Rule:** Don't headline technical architecture (MCP, dual surfaces, omni-triggers) — customers care about outcomes, not how it works
